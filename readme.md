# Instructions

## Api endpoints

```
GET all questions: http://127.0.0.1:8000/stack/questions/

GET single question: http://127.0.0.1:8000/stack/questions/<str:id>

POST : http://127.0.0.1:8000/stack/questions/

PUT : http://127.0.0.1:8000/stack/questions/<str:id>

DELETE : http://127.0.0.1:8000/stack/questions/<str:id>

Latest Questions : http://127.0.0.1:8000/stack/latest/

```
